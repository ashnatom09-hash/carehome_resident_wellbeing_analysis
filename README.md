# Care Home Residents Wellbeing Analysis

## Project Description
This project analyzes the wellbeing of care home residents using Python, SQL, and Power BI.  
It looks at physical, cognitive, and emotional health to provide insights for better care.

## Files in the Project
- **Python Notebook:** `CH_residents_wellbeing.ipynb` – data cleaning and analysis  
- **SQL Queries:** `queries.sql` – used to extract and summarize data  
- **Power BI Dashboard:** `CareHome_Wellbeing_Report.pbix` – interactive charts  
- **Reports:**  
  - `CareHome_Wellbeing_Report.pdf` – detailed report  
  - `CareHome_Wellbeing_Presentation.pptx` – presentation slides  

## How to Use
1. **Python Notebook:** Open in Jupyter, run cells to see tables, charts, and analysis.  
2. **SQL Queries:** Open in PostgreSQL to see data queries and results.  
3. **Power BI Dashboard:** Open `.pbix` file in Power BI Desktop to explore interactive charts.  
4. **PDF & PowerPoint:** Open to see the report and presentation summary.
